# Contributing to the PM Hub

How to propose changes
- Edit the file and open a PR (preferred) or open an issue for discussion.
- Use the PR template and include:
  - Summary of change
  - Which pages/templates are affected
  - Tests/examples (if a template)
  - Update front-matter: owner, last_reviewed (today), review_cadence

PR labels & reviewers
- Add label `docs/pm` for routine edits; `docs/pm-suggestion` for new content.
- Assign the owner listed in OWNERS.md; if none, assign `pm-maintainers`.

Quick PR checklist
- [ ] Front-matter populated (owner, last_reviewed, review_cadence)
- [ ] Tags added
- [ ] Example usage included (if template)
- [ ] Spell-checked and short (scannable)

Contact & office hours
See OWNERS.md for maintainers and schedule.
