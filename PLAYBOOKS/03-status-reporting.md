# Status Reporting — OctoAcme (Playbook)

Purpose
Provide consistent, scannable status updates for stakeholders.

Cadence
- Weekly updates for active projects; monthly for long-running maintenance.

Format
- Use TEMPLATES/weekly-status-update.md as the canonical format.
- Keep summary to 1–2 lines; use health (Green/Yellow/Red).

Distribution
- Send to project stakeholders and archive a copy in the project repo or shared drive.
