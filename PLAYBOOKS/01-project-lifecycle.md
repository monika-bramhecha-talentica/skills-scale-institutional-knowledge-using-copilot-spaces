# Project Lifecycle — OctoAcme (Playbook)

Phases
1. Initiate
   - Required artifacts: Project Charter (TEMPLATES/project-charter.md), Stakeholder list
   - Goal: align on objectives, success metrics, drivers
2. Plan
   - Required artifacts: Milestones & Owners, RAID summary
   - Goal: clear roadmap, release criteria, key dependencies
3. Deliver
   - Required artifacts: Weekly status updates, Decision log
   - Goal: track progress, manage impediments, escalate decisions
4. Close
   - Required artifacts: Retro, Postmortem (if applicable), Archive artifacts to EXAMPLES/
   - Goal: transfer learnings, document outcomes, tidy workspace

Required for every project
- A project charter created within first 2 weeks
- A RAID log maintained weekly
- Weekly status updates sent to stakeholders
